Requisitos por Sistema Operativo

Windows
  • Requiere Desktop Agent con privilegios de Administrador
  • Protección vía diskpart (write protect bit)
  • O vía registro: StorageDevicePolicies
Linux
  • Requiere permisos sudo o grupo disk
  • Proteger: sudo blockdev --setro /dev/sdX
  • Desproteger: sudo blockdev --setrw /dev/sdX
macOS
  • Requiere permisos de diskutil
  • Montaje solo lectura: diskutil mount readOnly diskX
  • SIP puede bloquear ciertas operaciones
