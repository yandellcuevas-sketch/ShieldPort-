ℹ️ Requisitos por Sistema Operativo

🪟
Windows
  • Requiere Desktop Agent con privilegios de Administrador
  • Protección vía diskpart (write protect bit en GPT/MBR)
  • O vía registro: HKLM\SYSTEM\CurrentControlSet\Control\StorageDevicePolicies
  • Comando: diskpart → select disk X → attributes disk set readonly
🐧
Linux
  • Requiere permisos sudo o grupo disk
  • Comando: sudo blockdev --setro /dev/sdX
  • Desproteger: sudo blockdev --setrw /dev/sdX
🍎
macOS
  • Requiere permisos de diskutil
  • Montaje solo lectura: diskutil mount readOnly /dev/diskX
  • SIP (System Integrity Protection) puede bloquear ciertas operaciones
